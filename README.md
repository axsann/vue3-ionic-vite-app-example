# Ionic Tab App (Vue 3 + Typescript + Vite) 

## Included Example
- Tab bar with vue-router
- LocalStorage by capacitor
- Global Store by provide/inject
- iOS / Material Design Page Transition Animation
- ionicons
- Modal
- Back button
- Can build iOS and Android app
